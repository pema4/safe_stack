# Safe Stack.
## Homework #3. for ISP RAS course.

## How to build

I used Ubuntu 20.04 within WSL

```bash
git clone --recursive https://github.com/pema4/safe_stack
cd safe_stack
cmake -B build
cmake --build build -t tests
cmake --build build -t main
build/app/main
```
